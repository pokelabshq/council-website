🔑 Keyword API

Extract keywords, entities, and text statistics. Free tier + x402 payments.

Python stdlib only 3 free reqs/day x402 USDC on Base TF-IDF + bigram scoring

Endpoints

POST /api/keywords — Extract keywords from text
POST /api/entities — Extract named entities
POST /api/analyze — Full analysis (keywords + entities + stats)
GET /api/health — Health check
GET /api/usage — Check your usage

Example Request

curl -X POST /api/keywords \
  -H "Content-Type: application/json" \
  -d '{"text": "Your text here...", "top_n": 10}'
