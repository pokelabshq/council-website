§ install

get started in 30 seconds

One line to install. One command to start.

# install
$ curl -fsSL https://raw.githubusercontent.com/pokelabshq/council/main/scripts/install.sh | bash

# start the setup wizard
$ pokelabs setup

# or jump straight in
$ pokelabs

Works on Linux, macOS, WSL2, and Termuv. Windows native support in early beta.

§ faq

questions

What model providers are supported?

OpenRouter (200+ models), Anthropic, OpenAI, Hugging Face, NovitaAI, NVIDIA NIM, z.ai/GLM, Kimi/Moonshot, MiniMax, Xiaomi MiMo, AWS Bedrock, Azure, or your own custom endpoint. Switch providers with a single command.

Is my data safe?

Council uses OAuth with minimal permissions. Your data stays on your infrastructure. The agent runs locally or on your own VPS — never on our servers. All code is open source and auditable.

Can I self-host?

Yes. Council is MIT licensed. The entire stack is open source. Run it on a $5 VPS, a GPU cluster, or serverless infrastructure. No vendor lock-in, ever.

What are the system requirements?

Python 3.11+ and a working terminal. That's it. For voice features you'll need a microphone. For the web dashboard you need a modern browser. Runs on Raspberry Pi, VPS, laptop, or cloud.

How does the learning loop work?

Council periodically nudges itself to persist important knowledge. After complex tasks, it autonomously creates reusable skills. These skills self-improve during use. FTS5 session search enables cross-session recall. Optional Honcho integration provides dialectic user modeling.
